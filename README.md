# Duma

Duma is a learning platform designed for accelerated education.

## Stack

- Java 21
- Spring Boot
- PostgreSQL
- MongoDB
- Redis
- RabbitMQ
- React / Next.js
- Docker

## Repositories

- duma-backend
- duma-ava
- duma-adm
- duma-infra

## Architecture

The system follows a modular monolith architecture with
event-driven components and asynchronous processing.
